Linux使用总结

发表于 2023-06-14 分类于 总结
本文字数： 1.2k 阅读时长 ≈ 1 分钟

前言

记录一下使用Linux系统过程中遇到的知识点

修改密码

passwd命令，然后输入当前的登录密码，再输入两次新密码即可。

快捷补全

按Tab键即可快捷补全，同Windows系统。

切换目录

cd命令，同Windows系统。

创建目录

mkdir Environment命令，创建一个名为Environment的目录。

查看工作目录下的内容

ls [-alrtAFR] [name...]

  • ls -l：以长格式的形式查看当前目录下所有可见文件的详细属性，其中rwx、r-x、r-x各有功能。
  • ls -a：显示当前目录中的所有文件和目录，包括隐藏文件。

清屏命令

  • clear：这个命令将会刷新屏幕，本质上只是让终端显示页向后翻了一页,如果向上滚动屏幕还可以看到之前的操作信息。
  • reset：这个命令将完全刷新终端屏幕，之前的终端输入操作信息将都会被清空，整个命令过程速度有点慢，使用较少。

监控GPU命令

nvidia-smi命令，同windows系统。

创建虚拟环境

virtualenv -p /usr/local/bin/python env1命令，创建一个名为env1的虚拟环境目录，/usr/local/bin/python是Python的安装路径，创建的虚拟环境中的Python版本与路径中的Python版本一致。

如何查询python安装路径

  • whereis python命令，可以在输出信息中查看出所有python的安装目录。
  • which python命令，可以查看出当前使用的python的安装目录。

激活虚拟环境

source env1/bin/activate命令，激活虚拟环境，激活成功之后命令行前面会出现(env1)。

退出虚拟环境

deactivate命令，退出虚拟环境。

删除虚拟环境

rm -rf env1命令，删除名为env1的虚拟环境。

安装GPU版本的PyTorch

建议先进入torch的wheel下载网站https://download.pytorch.org/whl/torch_stable.html查看版本对应的情况。拿我自己举例，服务器下安装的`CUDA`的版本是`12.1`，但是`PyTorch`的[官网](https://pytorch.org/get-started/locally/)上显示目前支持的`CUDA`版本为`11.7`和`11.8`，秉持着用新不用旧的原则，就打算安装官网提供的`11.8`的`Command`，但是没有安装成功，然后找了半天发现自己的`python`版本是`3.7`，在`wheel`网站里发现`3.7`版本的`python`最高支持的`CUDA`版本为`11.7`，于是在官网中的`Compute Platform这一项选择了CUDA 11.7，然后在命令行运行官网提供的Command，安装完成之后执行torch.cuda.is_available()，返回True`。
